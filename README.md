## 📖 Project Overview

The **Hospital Management System** is a relational database project developed using **MySQL** to simulate real-world hospital operations. It demonstrates database design, normalization, relationship management, and analytical SQL queries that provide meaningful business insights.

This project showcases practical SQL skills by managing patients, doctors, appointments, treatments, billing, and departmental data in a structured relational database.

---

## 🌟 Repository Highlights

* Designed a normalized relational database for hospital management.
* Established relationships using **Primary Keys** and **Foreign Keys**.
* Populated the database with realistic sample data.
* Developed SQL queries to answer business-oriented analytical questions.
* Organized the repository with clean documentation and modular SQL scripts.

## 🚀 How to Run

1. Clone the repository.

```bash
git clone https://github.com/sahanajayaseelanj-pixel/Hospital-Management-System-SQL.git
```

2. Open **MySQL Workbench**.

3. Execute the SQL files in the following order:

* `create_database.sql`
* `schema.sql`
* `sample_data.sql`

4. Run the queries inside:

```
analysis/business_queries.sql
```
