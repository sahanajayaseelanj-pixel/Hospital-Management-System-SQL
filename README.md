## 📖 Project Overview

This project demonstrates the design and implementation of a Hospital Management System using MySQL. It models real-world hospital operations through a relational database and includes analytical SQL queries to extract meaningful business insights.

The project showcases database design, normalization, data manipulation, and SQL-based reporting.

---

## 🎯 Objectives

- Design a normalized relational database
- Manage patient, doctor, and appointment records
- Store billing and medical information
- Perform business-oriented SQL analysis
- Practice advanced SQL concepts
